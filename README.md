# Saúde Masculina - App de Exercícios

App de exercícios para saúde masculina, focado em exercícios baseados em evidências científicas que ajudam na disfunção erétil.

## Design para público 45+

- **Cores chamativas**: azul naval (confiança) + âmbar/dourado (energia)
- **Alto contraste**: texto legível
- **Botões grandes**: mínimo 48px para facilitar toque
- **Navegação simples**: 3 abas principais
- **Interface limpa**: sem distrações

## Exercícios incluídos

1. **Assoalho Pélvico (Kegel)** - Os mais eficazes (até 75% de melhora em estudos)
2. **Yoga** - Melhora fluxo sanguíneo e relaxamento
3. **Cardiovascular** - Saúde do coração = saúde vascular
4. **Respiração** - Reduz ansiedade e melhora oxigenação

## Como rodar

```bash
npm install
npm run dev
```

Acesse http://localhost:5173 no navegador.

## Build para produção

```bash
npm run build
```

---

**Aviso**: Consulte um médico antes de iniciar qualquer programa de exercícios. Este app é um complemento, não substitui orientação médica.
